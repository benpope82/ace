Want a fantastic editor for Monstra CMS's theme editing pages in the admin panel? This is it.

What is Ace?

Ace is an embeddable text editor implemented in Javascript and made and shared by Cloud9. It provides syntax highlighting for various markup and scripting languages including HTML, CSS and Javascript making it an excellent choice for editing Monstra theme files via Monstra's Admin interface.

About the plugin

This plugin is an implementation of Ace for Monstra provided by MonstraHost.

Installation

  • Download the file and place the unzipped or untarred directory in your Monstra installation's 'plugins' directory.
  • Disable 'codemirror' because this plugin provides the same functionality
